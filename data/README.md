# Data Directory

This directory is **not tracked by git** (see `.gitignore`).

## Structure

```
data/
├── raw/           # Place Lending_club_data.csv here
└── processed/     # Auto-generated by the pipeline
    ├── scaler.joblib
    ├── X_train.parquet
    ├── X_test.parquet
    ├── y_train.parquet
    ├── y_test.parquet
    └── models/
        ├── xgboost.joblib
        ├── lightgbm.joblib
        └── random_forest.joblib
```

## Setup

Copy `Lending_club_data.csv` into `data/raw/` before running the pipeline.
